## 설명 📢

**1:1 채팅 로그를 기반으로 유튜브, 뉴스, 도서, 쇼핑 등 유저를 위한 맞춤 추천 컨텐츠를 제공하는 메신저 입니다.**<br/>
_컨텐츠를 찾고 공유하기 위해 앱을 이탈할 필요 없이 바로 컨텐츠를 검색하거나 채팅 내용에 기반한 추천 컨텐츠도 받아보세요!_

## 개발환경설정

```bash
# 가상 환경 활성화
$ pipenv --python 3.8.10
$ pipenv shell
```
```
# 패키지 설치
$ pipenv install
```
```
# 도커 실행
$ docker-compose up -d --build
```
```
# 테스트 실행
$ pipenv pytest
```
